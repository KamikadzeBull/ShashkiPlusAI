некоторая информация по совершению ходов

    поле работает по принципу
        1 клик - исходная клетка
        2 клик - конечная клетка
    если первым кликом выбрана неверная исходная клетка, вторым кликом просто сделайте неправильный ход,
    например, кликните на клетку, где уже расположена какая-то шашка
